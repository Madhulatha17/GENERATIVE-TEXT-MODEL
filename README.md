# DESCRIPTION OF TASK

The objective of this task is to build a Generative Text Model that can generate coherent and meaningful text based on a given user prompt.

In this task, we implement a text generation system using deep learning techniques such as LSTM (Long Short-Term Memory) networks or GPT-2 transformer models. The model is trained on sample text data so that it learns

patterns, structure, and relationships between words.

Once trained, the model is able to:

Accept a user input (prompt or topic)

Predict the next words sequentially

Generate meaningful sentences or paragraphs related to the input
